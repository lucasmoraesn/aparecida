## 🆘 Suporte & Troubleshooting

### Problema: Página não carrega
```
Solução:
1. npm install
2. npm run dev
3. Limpar cache (Ctrl+Shift+Delete)
```

### Problema: WhatsApp não abre
```
Solução:
1. Verificar número (55 + código área + número)
2. Verificar codificação da URL
3. Testar em desktop primeiro
```

### Problema: Componente não renderiza
```
Solução:
1. Verificar import: import Component from '../path'
2. Verificar fechamento de tags
3. Revisar console.log (F12)
```

### Problema: Estilos não aparecem
```
Solução:
1. npm install tailwindcss (se necessário)
2. Limpar build: npm run build
3. Reiniciar dev: npm run dev
```
